# BAD703: Data Security and Privacy

Welcome to the official course homepage for **BAD703 – Data Security and Privacy**.

---

## 📘 Course Overview

In an era where data is the new oil, securing it and preserving privacy have become foundational competencies for any data professional. This course equips students with both the **theoretical foundations** and **practical tools** needed to design, implement, and audit secure, privacy-preserving data systems.

Students will explore cryptographic primitives, access control models, privacy-enhancing technologies (PETs), regulatory frameworks (GDPR, HIPAA, India's DPDP Act), and modern threats such as data breaches, inference attacks, and AI-driven privacy risks.

## 🧭 Course Objectives

- Understand fundamental concepts of data security, privacy, and the CIA triad.
- Apply cryptographic techniques to protect data at rest, in transit, and in use.
- Analyze access control, authentication, and identity management frameworks.
- Evaluate privacy-preserving techniques: anonymization, differential privacy, k-anonymity.
- Interpret global data protection laws and design compliant data systems.
